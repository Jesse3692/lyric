# 歌词生成模型

这个项目实现了一个基于深度学习的歌词生成模型，可以根据不同风格（歌手、地区等）生成歌词。

## 功能特点

- 数据预处理：清洗歌词语料，只保留中英文和标点符号
- 模型训练：支持训练单一模型或多种风格的模型
- 歌词生成：根据提示文本生成不同风格的歌词
- 网页界面：使用Gradio提供交互式歌词生成界面

## 安装依赖

```bash
pip install -r requirements.txt
```
